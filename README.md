## Using this dependency

**Maven**:

```xml
<dependency>
    <groupId>com.jacobtread.netty</groupId>
    <artifactId>kotlin-netty-http</artifactId>
    <version>{VERSION}</version>
</dependency>
```

**Groovy**:

```groovy

repositories {
    mavenCentral()
}

dependencies {
    implementation 'com.jacobtread.netty:kotlin-netty-http:{VERSION}'
}
```

**Kotlin DSL**:

```kotlin

    mavenCentral()
```
